# Corivo · 设计文档 v0.5

> **数字孪生系统** —— 你的跨平台数字自我，在每个工具中认识你、学习你、代表你。

---

## Corivo 是什么

Corivo 是一个嵌入用户已有工作流的**数字孪生系统**。它不是一个独立的 App，而是寄生在 Claude Code、Cursor、飞书等工具中的后台服务——自动从用户的 AI 对话和消息中学习决策模式、理解偏好、积累上下文，在合适的时机以 `[corivo]` 的名义主动提供建议。

用户不需要下载新软件、学习新界面、改变任何习惯。安装一个 CLI，跑一次 `corivo init`，数字孪生就开始成长了。

---

## 解决什么问题

**你的数字自我是碎片化的。**

你在 Claude 里说的技术偏好，Cursor 不知道。你在飞书里达成的共识，ChatGPT 不知道。你在某个项目里做过的决策，新项目里要重新讨论一遍。

Corivo 把这些散落的数字足迹自动汇聚成一个统一的**数字孪生**——不只是记住信息，更是学习你的决策模式、预测你的选择、代表你的立场。

---

## 核心理念

**从"记忆层"到"数字孪生"**

- v0.2：个人记忆层 —— 采集、存储、检索
- v0.5：数字孪生系统 —— 学习、预测、代理

**文本是唯一的"源码"。** Agent 能处理，人能直接读。所有结构化索引都是文本的派生物。

**人是主人，Agent 是管家。** 记录、整理、学习、预测——全部由 Agent 执行。人只在需要时阅读结果。

**自然语言即编程语言。** 用户说一句"以后周报帮我这样写"，Agent 将其存为指令，下次自动执行。

**主动训练 + 被动学习。** Corivo 从你的行为中学习，也允许你主动告诉它你的偏好。

**安全是基础。** 所有数据只存在用户本地设备上，端到端加密同步。Corivo 团队无法读取用户数据。项目开源，可审计、可自托管。

---

## 核心流程

```
采集（无感）──→ 学习（自动）──→ 预测（主动）
                    ↑                  │
              记忆生命周期              │
              （衰减/整合/重构）        │
                                      ↓
        ← 用户在已有工具中自然交互 ←──┘
```

**采集**：通过规则注入让 AI 工具主动存储信息，通过事件订阅监听消息平台。用户无感。

**学习**：心跳引擎持续处理新数据、整合重复内容、提取决策模式、学习用户偏好。用户无感。

**预测**：在对话中以 `[corivo]` 的名义主动提供建议，基于历史模式预测用户的选择。用户有感。

---

## 产品定位

Corivo 是跨工具数字自我的基础设施——**你的数字身份不属于任何单一工具**。

你在飞书里达成的共识，影响 Cursor 给你的代码建议。你在 ChatGPT 里表达的观点，出现在 Claude 的上下文中。

**护城河**：当 OpenAI 给 ChatGPT 加长期记忆、Anthropic 给 Claude 加跨会话记忆时，Corivo 的价值在于**跨平台的连续性**。

## v0.5 新增能力

### 1. 决策模式学习

Corivo 不只存储你的决策，还提取决策背后的模式：
- 技术选型的偏好维度（安全 > 便利，成本 > 性能）
- 沟通风格（正式/随意，直接/委婉）
- 决策流程（数据驱动/直觉导向，快速迭代/深思熟虑）

### 2. 预测性建议

基于学习到的模式，Corivo 可以预测你在新情境下的选择：

```
用户：我们要为新项目选后端框架

[corivo] 预测：基于你过去 6 个月的技术选型模式（7/8 优先考虑 TypeScript，5/5 选择本地优先存储），我建议你考虑：
- Deno（内置 TypeScript）
- Supabase（本地开发友好）

要我看一下其他类似的决策记录吗？
```

### 3. 主动训练

用户可以主动告诉 Corivo 自己的偏好：

```bash
$ corivo train "技术选型优先级：安全 > 成本 > 性能"
$ corivo train "沟通风格：对技术团队直接，对非技术团队用类比"
$ corivo train "周报格式：用数字说话，列问题带方案"
```

训练后的偏好会被标记为 `指令 · self · 训练`，Prediction Agent 优先参考。

## 技术选型概览

| 层面 | 选型 | 理由 |
|------|------|------|
| 本地存储 | SQLCipher（加密 SQLite） | 全平台、毫秒级、离线可用 |
| 语义搜索 | sqlite-vss / 向量嵌入 | 个人数据量级足够 |
| 模式学习 | LLM 提取 + 结构化存储 | 平衡准确性和成本 |
| 多设备同步 | E2EE 中继服务 | 安全等价 P2P，体验等价云同步 |
| 对外接口 | CLI（核心）+ MCP Server + IM Bridge | 覆盖 CLI Agent、GUI AI 工具、消息平台 |
| 心跳运行 | MCP 宿主 / 独立守护进程（可配置） | 默认寄生，可选独立 |
| 预测生成 | 本地模型优先，API 备选 | 隐私 + 准确性 |
